✈️ Flight Analytics Dashboard
📌 Project Overview

This project is an end-to-end Flight Analytics System that collects aviation data from external APIs, stores it in a SQL database, performs analytical queries, and visualizes insights through an interactive Streamlit dashboard.

The application enables users to explore airport information, flight operations, aircraft details, delay statistics, and route performance in a user-friendly interface.

🎯 Project Objective

The main objective of this project is to:

Extract real-time aviation data using APIs.
Store and manage data using SQLite.
Perform SQL-based analysis on flights, airports, aircraft, and delays.
Build an interactive dashboard using Streamlit.
Generate operational insights through visualizations and reports.
